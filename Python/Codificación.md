# Hash

Función que transforma datos de longitud variable en una cadena de longitud fija, diseñada para ser única y no reversible

Determinismo: Misma entrada->misma salida
Resistencia a colisiones: Dificil encontrar dos entradas co el mismo hash
Resistencia a preimagen: Muy dificil deducir la entrada a partir del hash
Avalancha: Pequeños cambios en la entrada->grandes cambios en el hash
